# WebComponents
Charla realizada en la Universidad de Cantabria, en las jornadas UCTechPro

Este proyecto contiene
- slides usadas en la charla
- proyecto Biblioteca

## Instalacion
### Dependencias npm
```
npm install
```
### Dependencias bower
```
bower install
```

### Arrancar Servidor express
``` 
gulp
```
